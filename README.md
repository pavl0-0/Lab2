Лабораторна робота №2. Розробка серверного застосунку на Node.js з використанням Express та React. Цей проєкт являє собою повноцінний веб-застосунок, розроблений у форматі інтерактивного IT-блогу для демонстрації навичок роботи з сучасним веб-стеком та архітектурою клієнт-сервер. Додаток дозволяє користувачам проходити реєстрацію та авторизацію, створювати власні публікації, залишати коментарі, а також швидко знаходити потрібний контент завдяки системі категорій та динамічному текстовому пошуку. Для забезпечення максимальної швидкодії під високими навантаженнями реалізовано кешування результатів запитів на рівні сервера, що суттєво мінімізує кількість звернень до бази даних. Уся інформація надійно зберігається у реляційній базі SQLite, що гарантує збереження даних навіть після перезапуску сервера. Клієнтська частина написана на React з використанням збирача Vite та має сучасний адаптивний дизайн із темною темою, а серверна частина побудована на Node.js та фреймворку Express. Проєкт розділено на дві незалежні директорії: server та client. Для його роботи потрібно запустити обидві частини у двох різних вікнах терміналу. Спочатку необхідно перейти в папку server, встановити залежності командою npm install та запустити бекенд командою node app.js. Сервер розпочне роботу на порту 3000, і при першому запуску автоматично створить файл бази даних database.sqlite разом із базовими категоріями. Після цього у новому вікні терміналу потрібно перейти в папку client, також встановити залежності через npm install і запустити клієнтську частину командою npm run dev. Застосунок буде доступний для використання у браузері за локальним посиланням, яке з'явиться у терміналі.
